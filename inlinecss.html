Cascading Style Sheete (CSS)

CSS is used to style and describe the layout of a web page

CSS can be added to a webpage in three formats namely;

  1. Inline CSS
  2. Internal CSS
  3. External CSS

CSS has some of the following functions in a webpage

  • Changing the color of elements
  • Adjusting the size and dimension of elements
  • Adjusting the layout of elements
